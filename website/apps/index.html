Mailpile

Hacker CMS

Developer: Kenton Varda | Last Updated: ____

This is a mash-up of Jekyll, Ace, and jsTree to create a Markdown-based web site editing and publishing app. This app was stiched together over the course of one day by Kenton.

License: 2-Clause BSD | Source: Github

IPython Notebook

Developer: Jason Paryani | Last Updated: ____

A port of IPython Notebook, a web-based interactive computational environment where you can execute Python code, plot graphs, render equations, and more.

License: 3-Clause BSD | Source: Github

Meteor Todo List

Developer: Kenton Varda | Last Updated: ____

This is a port of the "Todo List" example application that comes with the Meteor development tools. No code changes were necessary.

License: MIT | Source: Github

Web Shell

Developer: Kevin Wallace | Last Updated: ____

A very simple front-end to a Busybox shell. Can be used to poke at the Sandstorm sandbox. (Note: As of this time, resource-exhaustion-based DoS attacks are easy to carry out from a shell. We're working on tightening things up. If you are a Sandstorm Alpha user, please be nice and don't crash our server, thanks. :))

License: MIT | Source: Github

Duoludo

A minimalist platformer game with checkpoints, replay, and collaboration. A port of duoludo.

License: 3-Clause BSD | Source: Github

Acronymy

Developer: David Renshaw | Last Updated: ____

A little game / tech demo. The objective is to define every English word as an acronym of other words. The app is notable in that it is written in Rust directly against the low-level Sandstorm API, thus avoiding the need for an HTTP server.

Also try David's shared instance.

License: 2-Clause BSD | Source: Github
